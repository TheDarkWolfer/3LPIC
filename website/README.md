# Coursero website

## Usage

This website allow users to authenticate themselves with an email address. Then they will be able to select a course, an exercise and the programming language. The result of the exercise is visible once the test is finished.

## Starting up

**Local usage**

To start the website, go to the project root with ```cd website``` and then ```npm run dev```

**Server usage**

To host on server, go to the project root with ```cd website``` and then ```npm run build```

## Technologies and architecture

This project is made with React native and is thinked to be hosted with reverse proxy
